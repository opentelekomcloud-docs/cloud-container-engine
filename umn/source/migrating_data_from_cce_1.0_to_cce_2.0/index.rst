Migrating Data from CCE 1.0 to CCE 2.0
======================================

Perform the following operations to migrate data from CCE 1.0 to CCE 2.0:

#. Migrating Images: Pull images from the image repository of CCE 1.0, and push them to the image repository of CCE 2.0.
#. Migrating clusters: Create Hybrid clusters in CCE 2.0 for application deployment.
#. Migrating applications: Move applications deployed in CCE 1.0 to the new clusters of CCE 2.0.
